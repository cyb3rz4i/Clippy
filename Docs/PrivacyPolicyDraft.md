# Clippy Privacy Policy Draft

Clippy stores clipboard history locally on your Mac. Clipboard entries may include text, links, and images you copy after enabling clipboard history.

Clippy does not create accounts, sync data to cloud services, sell data, share data with third parties, use analytics, use advertising SDKs, or require network access.

Clipboard history is stored in the app's local container. You can delete individual items, clear all history, pause capture, disable capture, or exclude specific apps from capture.

If you enable auto-paste, macOS may ask you to grant Accessibility permission. Clippy uses that permission only to send the paste command after you choose a clipboard history item. Auto-paste is optional.

If you enable launch at login, macOS starts Clippy when you sign in. You can turn this off in Settings.
